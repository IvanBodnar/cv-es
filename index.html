Experiencia Laboral

Observatorio de Seguridad Vial GCBA | 2013 - 2017

Comencé en el Observatorio de Seguridad Vial administrando los datos sobre siniestros viales, sobre todo su componente geográfico, a través de ArcGis.
Con el tiempo empecé a aprender por mi cuenta Python para poder automatizar tareas de ArcGis.
Al ir aprendiendo sobre bases de datos relacionales (Postgres específicamente), normalicé los datos e implementé un servidor Linux, usando Postgres y PostGis para manejar los datos geográficos.
Con Python desarrollé varias aplicaciones usando PyQT, para permitir que el personal no técnico pudiera ingresar datos a la base, asi como también disparar tareas de geoprocesamiento y extraer datos solicitados por otras reparticiones. Luego fui aprendiendo desarrollo web con Django, lo cual me permitió hacer una aplicación web para que distintas reparticiones dentro del GCBA pudieran consultar datos sobre los siniestros viales. Esta aplicación se encuentra todavía en uso.
Cuando me dí cuenta de que no había posibilidades de crecer como desarrollador en el GCBA, busqué trabajo en el ámbito privado y renuncié.


Anachronics SRL | Noviembre 2017 - Agosto 2018

Anachronics se dedica a proveer soluciones informáticas para empresas de distribución de energía eléctrica y empresas de distribución de gas natural. Allí me desempeñé como desarrollador backend, usando plpgsql, SQL, y apis del sistema propietario que la empresa tiene implementado.
Mis tareas incluían implementar requerimientos, arreglar bugs, y mantener reuniones con los clientes para delinear el desarrollo de proyectos.
Luego de un tiempo me dí cuenta de que quería dedicarme al desarrollo web, por lo cual tomé la decisión de renunciar para aprender más sobre JavaScript, Angular y Python, y así poder conseguir otro trabajo.
